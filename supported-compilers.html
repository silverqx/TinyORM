Supported Compilers

Following compilers are backed up by GitHub Action workflows (CI pipelines) which also include more then 800 unit tests.

Windows >=10:

  • MSVC 2019 16.9 - 16.11.7
  • MSYS2 UCRT64 GCC 10.2 - 11.2
  • MSYS2 UCRT64 Clang 12 - 13

Linux:

  • GCC 10.2 - 11.2
  • Clang 11 - 13
